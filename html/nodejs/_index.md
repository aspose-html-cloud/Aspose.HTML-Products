<p>Aspose.HTML Cloud SDK for Node.js makes your Node.js cloud-based applications able to access HTML documents from cloud storage and render them in their native format or any of the other supported formats, e.g., PDF, XPS, JPEG, TIFF, PNG, etc.</p>
<p>Using Aspose.HTML Cloud SDK your Node.js programs can build HTML pages by inserting data into predefined templates. You can insert data in the HTML document template via request body or by loading data from a local file.</p>
<p>Aspose.HTML Cloud SDK is built on top of HTML REST API. Our Cloud SDK is offered under MIT license. So you can utilize it within your cloud-based Node.js applications easily and seamlessly.</p>
<p>Aspose.HTML Cloud SDK for Node.js allows your Node.js code to get a list of matching HTML fragments using XPath query or CSS selector. Using this Node.js Cloud SDK you can easily convert HTML to JSON, XHTML, HTML containing SVG Markup, Markdown, PDF, XPS, TIFF, GIF, ZIP (images) and other supported file formats.</p>
